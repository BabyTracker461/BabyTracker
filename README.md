# Baby Tracker

## Get it running

1. Install dependencies

   ```bash
   npm install
   ```

2. Start supabase

   ```bash
   npx supabase start
   ```
    - make sure you do not use localhost as the server runs on your computer, not your phone, instead use your computer local ip

3. Start the app

   ```bash
    npx expo start
   ```
